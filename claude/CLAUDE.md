# Global Claude Instructions

## Language & Communication

- 討論用台灣繁體中文，commit message 與專有名詞用英文
- 預設簡短直接，有問題我會主動追問
- 不要當 yes man——如果覺得我的方向有問題，直接提出不同意見

## Working Style

- 非 trivial 的任務先從 plan mode 開始，對齊方向再動手
- 最小變更——不要做超出需求的 refactor、不加多餘註解或 docstring

## Proactivity

- 積極給改善建議，我會自己篩選
- 明顯的 bug/typo 可以順手指出
- 但大改動先提出來讓我決定
